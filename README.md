# skymu.app

This repository holds our website, as seen at https://www.skymu.app. A major rewrite of the entire site has recently been completed. The website's creator/copyright holder is OmegaAOL and it is licensed under the Affero General Public License, version 3 or later.

Any non-open source files hosted on skymu.app are separate from the website and intentionally excluded from this repository. The website files in this repository are statically hosted on https://theskymuteam.github.io for testing purposes.

## Browser compatibility
* Google Chrome 1.0 and above.
* Mozilla Firefox 2.0 and above.
  * Older versions untested
* Internet Explorer 7 and above.
  * IE6 has not been tested, IE5 is confirmed to have layout bugs.
* Netscape Navigator 9.0 and above.
  * Older versions untested
* Opera 12.0 and above.
  * Older versions untested
